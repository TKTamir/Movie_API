Movies API

Hi! welcome to my Movies API created by Tamir Kahalany.
This is a REST API connected to a database that stores data abot different movies.

Movies API actions

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of all movies /movies GET none JSON object containing data-Description, genre, director, image URL, featured status
Return data about a signle title /movies/:title GET none JSON object containing data about a title-Description, genre, director, image URL, featured status
Return data about a genre /movies/genre/:genreName GET none JSON object containing data about a genre
Return data about a director /movies/directors/:directorName GET none JSON object containing data about a director-Name, bio, birth year, death year
Register new user /users POST { id: , name: "", password:"", favoriteMovies: [] } JSON object containig data about a user:
{ id: , name: "", password:"", favoriteMovies: [] }
Update user data /users/:id PUT none JSON object containig data about a user:
{ id: , name: "", password:"", favoriteMovies: [] }
Update user favorite movie /users/:id/movieTitle POST none JSON object containig data about a user:
{ id: , name: "", password:"", favoriteMovies: [] }
Remove user's favorite movie /users/:id/movieTitle DELETE none Text confirming that a movie was removed
Remove user /users/:id DELETE none Text confirming that a user was removed
